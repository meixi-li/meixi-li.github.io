This work develops a quasiconformal geometry framework for craniofacial analysis using 3D facial surface data.

### Abstract

Obstructive sleep apnea (OSA) is a common sleep disorder that can be associated with craniofacial morphology. In this work, we introduce a quasiconformal-based geometric framework for analyzing craniofacial structures from reconstructed 3D facial surfaces. By combining Teichmüller shape analysis, curvature-based descriptors, and machine learning techniques, the proposed approach extracts discriminative facial features for OSA classification.

### My Contributions

- Developed feature extraction and classification pipelines.
- Computed geometric descriptors from reconstructed facial surfaces.
- Performed feature selection using bagging-based statistical methods.
- Evaluated classification performance across multiple feature representations.
- Contributed to manuscript preparation and validation.
